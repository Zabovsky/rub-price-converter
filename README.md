# Цена в рублях

Браузерное расширение для **Chrome** и **Firefox**, которое находит на страницах цены в иностранных валютах и показывает рядом эквивалент в рублях.

Пример: `1 200 ₫` → `1 200 ₫ (~4,50 ₽)` · `20.990.000đ` → `20.990.000đ (~63 000 ₽)`

## Возможности

- Автоматическое распознавание цен в USD, EUR, CNY, VND, HKD, JPY и других валютах
- Поддержка вьетнамского формата: `20.990.000đ` = 20 990 000 донгов
- Источники курса на выбор:
  - **ЦБ РФ** — официальный курс ([cbr-xml-daily.ru](https://www.cbr-xml-daily.ru/))
  - **Рыночный курс** — агрегированный курс ([open.er-api.com](https://open.er-api.com/))
  - **Московская биржа** — USD/EUR с MOEX, остальные валюты — по ЦБ РФ
- Включение и отключение в один клик
- Исключение отдельных сайтов (blacklist)
- Работа на динамических страницах (SPA)

## Быстрая установка

### Chrome / Edge / Brave

1. Скачайте или клонируйте репозиторий
2. Установите зависимости и соберите проект:

```bash
npm install
npm run build
```

3. Откройте `chrome://extensions`
4. Включите **Режим разработчика**
5. Нажмите **Загрузить распакованное расширение**
6. Выберите папку `dist/`

### Firefox

```bash
npm install
npm run build:firefox
```

1. Откройте `about:debugging#/runtime/this-firefox`
2. Нажмите **Загрузить временное дополнение…**
3. Укажите файл `dist-firefox/manifest.json`

> В Firefox временное дополнение удаляется после перезапуска браузера. Для постоянной установки можно подписать расширение через [addons.mozilla.org](https://addons.mozilla.org/) (пока не опубликовано).

## Использование

1. Нажмите на иконку расширения в панели браузера
2. Убедитесь, что переключатель **включён**
3. Выберите **источник курса**
4. Откройте любой сайт с ценами в иностранной валюте — рядом появится `(~X ₽)`

Для проверки откройте файл [`test-page.html`](test-page.html) в браузере.

## Разработка

### Требования

- Node.js 18+
- npm 9+

Если `npm install` зависает на `registry.npmjs.org`, в проекте уже настроено зеркало в [`.npmrc`](.npmrc).

### Команды

| Команда | Описание |
|---------|----------|
| `npm install` | Установка зависимостей |
| `npm run dev` | Сборка в watch-режиме |
| `npm run build` | Сборка для Chrome → `dist/` |
| `npm run build:firefox` | Сборка для Firefox → `dist-firefox/` |

### Структура проекта

```
src/
  background/     # service worker — загрузка и кеш курсов
  content/        # сканирование страниц и вставка цен в ₽
  popup/          # настройки расширения
  shared/         # парсер, конвертер, провайдеры курсов
```

## Ограничения

- Курсы кешируются на ~45 минут
- MOEX публикует только USD и EUR; для остальных валют используется ЦБ РФ
- Расширение не заменяет цену на сайте, а показывает рубли **рядом** с оригиналом
- Точность зависит от выбранного источника курса
